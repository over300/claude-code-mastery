# 개발자 웹 이력서

HTML, CSS, JavaScript, TailwindCSS를 활용한 반응형 개발자 포트폴리오 웹사이트입니다.

## 기술 스택

- **마크업**: HTML5
- **스타일링**: CSS3 + TailwindCSS
- **상호작용**: JavaScript (ES6+)
- **배포**: GitHub Pages / Netlify

## 프로젝트 구조

```
project/
├── index.html                # 메인 이력서 페이지
├── css/
│   └── style.css            # 커스텀 스타일
├── js/
│   └── script.js            # 상호작용 및 기능
├── assets/
│   ├── images/              # 이미지 자산
│   └── files/               # 다운로드 파일 (PDF 등)
├── ROADMAP.md               # 프로젝트 개발 로드맵
└── README.md                # 이 파일
```

## 시작하기

### 필수 조건
- 최신 웹 브라우저 (Chrome, Firefox, Safari, Edge)
- 선택사항: Python (로컬 서버 실행 시)

### 로컬 개발

```bash
# 로컬 서버 시작
npm run dev
# http://localhost:8000 에서 확인
```
