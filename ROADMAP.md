# 개발자 웹 이력서 개발 로드맵

## 프로젝트 개요
HTML, CSS, JavaScript, TailwindCSS를 이용한 반응형 개발자 웹 이력서 개발

## 기술 스택
- **마크업**: HTML5
- **스타일링**: CSS3 + TailwindCSS
- **상호작용**: JavaScript (ES6+)
- **배포**: GitHub Pages / Netlify (선택사항)
